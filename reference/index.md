# Package index

## GlobalArchive API

Essential functionality for accessing data through the GlobalArchive
API.

- [`ga_api_set_token()`](https://globalarchivemanual.github.io/GlobalArchive/reference/ga_api_set_token.md)
  : Set GlobalArchive API Token
- [`ga_api_all_data()`](https://globalarchivemanual.github.io/GlobalArchive/reference/ga_api_all_data.md)
  : Retrieve all Data from a synthesis using the GlobalArchive API
- [`ga_api_count()`](https://globalarchivemanual.github.io/GlobalArchive/reference/ga_api_count.md)
  : Retrieve Count Data from the GlobalArchive API
- [`ga_api_habitat()`](https://globalarchivemanual.github.io/GlobalArchive/reference/ga_api_habitat.md)
  : Retrieve Habitat Data from the GlobalArchive API
- [`ga_api_relief()`](https://globalarchivemanual.github.io/GlobalArchive/reference/ga_api_relief.md)
  : Retrieve Relief Data from the GlobalArchive API
- [`ga_api_length()`](https://globalarchivemanual.github.io/GlobalArchive/reference/ga_api_length.md)
  : Retrieve Length Data from the GlobalArchive API
- [`ga_api_metadata()`](https://globalarchivemanual.github.io/GlobalArchive/reference/ga_api_metadata.md)
  : Retrieve Metadata from the GlobalArchive API
- [`ga_api_species_list()`](https://globalarchivemanual.github.io/GlobalArchive/reference/ga_api_species_list.md)
  : Retrieve Species List from the Global Archive API
- [`ga_api_benthic_list()`](https://globalarchivemanual.github.io/GlobalArchive/reference/ga_api_benthic_list.md)
  : Retrieve Benthic Species List from the Global Archive API
